# CS1666-First-Team

CS 1666 Principles of Computer Game Design and Implementation

Fall 2019

# Set Up
Run `sh setup.sh` to install the following libraries
* SDL2
* SDL2 Image
* SDL2 Mixer
* sndio 6.1
* mad0
* madlib

# Running the Game
1. Run `sh build_game.sh` to build the executable
2. Run `./build/bin/Game` to run the executable
